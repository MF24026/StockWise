# StockWise

StockWise es un sistema de gestión de inventarios que permite a las empresas
controlar productos, proveedores y movimientos de stock de manera eficiente.

El sistema permitirá registrar entradas y salidas de inventario, visualizar
productos con bajo stock y llevar un historial de movimientos para mejorar
la toma de decisiones.

Proyecto desarrollado como práctica académica utilizando:

- Spring Boot
- React + TypeScript
- PostgreSQL
- Docker
