# Guía de Despliegue Local - StockWise

Este archivo registra las pruebas de entorno de desarrollo y despliegue para el Laboratorio 3.

## Pasos de Ejecución:
1. Verificación de Node.js v24+
2. Instalación de dependencias del ecosistema Vite via \
pm install\\
3. Levantamiento del servidor local en puerto 5173 via \
pm run dev\\
